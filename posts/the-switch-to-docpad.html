The Switch to DocPad

Written by Eduán Lávaque on 15 June 2013.

Well I finally switched to something else than WordPress.

I switched to DocPad, the Node.js based static website generator. It's a static website generator but if you wish it can be completely dynamic.

I am using Markdown to write my posts. Using CoffeeKup and Eco as templating languages and normal JavaScript for the .js files. And I am using SCSS for the CSS, of course.

Also, now my website is being hosted by GitHub Pages, so that's cool.

That is all. Hope you like the changes. :)
